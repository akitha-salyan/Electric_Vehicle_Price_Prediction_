****Electric Cars Price Prediction****

___________________________________________________
**Project Overview**

This project focuses on analyzing electric vehicle (EV) data and building a machine learning model to predict the price of electric cars. The system uses real EV datasets and applies data preprocessing, visualization, and regression modeling techniques to estimate car prices.

The final trained model is deployed using a simple web application interface.

_________________________________________________________________________________________________________________________________________

**🗂 Dataset Information**

The dataset contains information about electric vehicles such as:

- Car Name
- Manufacturer
- Battery Capacity
- Range
- Charging Time
- Power
- Acceleration
- Price

**The dataset is used for:**
- Data Cleaning
- Model Training

___________________________________________________________________________________________________________________________________________
**Machine Learning Model**
- The project uses regression algorithms to predict EV prices.

**Steps Involved:**

- Data Cleaning

- Handling Missing Values

- Feature Selection

- Train-Test Split

- Model Training

- Model Evaluation

- Saving the Model using Pickle

- The trained model is saved as:

    **ev_model.pkl**
________________________________________________________________________________________________________________________________________
  **🌐 Web Application**

- The model is deployed using Flask.
**Features:**
- User inputs EV specifications
- Model predicts estimated price
- Simple and interactive UI
- To run the application:
   python app.py
____________________________________________________________________________________________________________________________________________
  📊 Technologies Used

    - Python
